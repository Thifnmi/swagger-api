# IAM-document-v2

### Running the server
To run the server with python3:

1. Install dependencies: 
```
pip install -r requirements.txt
```
2. Run the server:
```

python3 server.py
```

To run the server with docker:
1. switch to root directory of project and run:
```
docker-compose up --b -d
```
